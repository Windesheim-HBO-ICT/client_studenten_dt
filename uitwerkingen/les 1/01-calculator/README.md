# Calculator

## Algemene afspraken
- Het is niet noodzakelijk en verplicht om jQuery te gebruiken
- Het gebruik van var wordt gezien als bad practice

## Stappen CSS
- Controleer inladen van css bestand in browser en maak het bestand werkend
- Schakel css in index.html in
- Verplaats css van index.html naar style.css
- Speel met flexbox in de browser
- Maak een uitstapje naar 02-flexbox index.html en speel met flexbox

## Stappen Javascript
- Controleer inladen van js bestand in de browser en maak het bestand werkend
- Schakel js in index.html in en controleer werking in de console (werkt niet!)
- Verplaats de javascript, zodat deze onder de DOM elementen staat. Controleer de werking opnieuw (werkt wel!)
- Verplaatst de javascript naar app.js en controleer de werking.
- Probeer de showFunction in de console uit.
- Zorg ervoor dat een klik op een knop ervoor zorgt dat het getal in het display wordt getoond.

## JSON en localstorage
- Maak een uitstapje naar 03-localstorage indext.html. Test de functies in de console en in localstorage.
- Voeg een functies toe om JSON objecten op te slaan, op te halen en te verwijderen in localstorage.
- Zorg ervoor dat het resultaat van de calculator opgeslagen wordt in localstorage.
